Markdown Cheatsheet

Headings

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Text Formatting

**bold** or __bold__
*italic* or _italic_
***bold italic***
~~strikethrough~~
`inline code`

Lists

- Item 1
- Item 2
  - Nested item
  
1. First
2. Second
3. Third

Links and Images

[Link text](https://example.com)
[Link with title](https://example.com "Title")
![Alt text](image.jpg)
![Alt text](image.jpg "Image title")

Code Blocks

```
code block
```

```python
# Python code block
print("Hello")
```

Blockquotes

> This is a quote
> It can span multiple lines
>> Nested quote

Horizontal Line

---
or
***
or
___

Tables

| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |

Escaping Characters

\*escaped asterisk\*
\[escaped bracket\]

Line Breaks

Line 1  
(two spaces at end)
Line 2

(or blank line between paragraphs)

Task Lists

- [x] Completed task
- [ ] Incomplete task
